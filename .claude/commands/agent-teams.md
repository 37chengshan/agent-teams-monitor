# Agent Teams Monitor

启动 Agent Teams Monitor 监控面板。

## 使用方法

在 Claude Code 中输入：
```
/agent-teams [命令]
```

## 命令

| 命令 | 说明 |
|------|------|
| start | 启动监控面板（前后端） |
| server | 仅启动后端 |
| client | 仅启动前端 |
| stop | 停止所有服务 |
| help | 显示帮助 |

## 访问地址

- 前端：http://localhost:3000
- 后端：http://localhost:3002

## 实现

运行 `.claude/commands/agent-teams.py` 脚本
