# Agent Teams Skills

本目录包含与 Agent Teams 相关的 Claude Code Skills。

## agent-teams-playbook

多 Agent 协作编排手册，提供完整的团队组建和协作指南。

### 安装

```bash
# 复制到 Claude Code skills 目录
cp -r agent-teams-playbook ~/.claude/skills/
```

### 使用

在 Claude Code 中使用以下任一触发词：

- "拉团队帮我[任务描述]"
- "多agent协作"
- "agent编排"
- "并行agent"

### 功能

- 5 大编排场景
- 6 阶段工作流
- Skill 自动发现
- 质量把关机制

详见 [agent-teams-playbook/SKILL.md](./agent-teams-playbook/SKILL.md)
